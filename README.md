# 📚 C# OOP Projects

A collection of Object-Oriented Programming (OOP) projects developed using C# and .NET.
This repository demonstrates practical implementation of core and advanced OOP concepts through multiple console-based systems and applications.

---

# 🚀 Concepts Covered

* Encapsulation
* Inheritance
* Polymorphism
* Abstraction
* Interfaces
* Partial Classes
* Extension Methods
* Composition
* Arrays & Collections
* File Organization & Clean Architecture

---

# 📂 Repository Structure

Each project is organized in a clean structure where:

* Classes are separated into individual files
* Interfaces are separated into dedicated files
* Partial classes are split logically
* Extension methods are placed in separate files
* Program entry point is isolated for readability
